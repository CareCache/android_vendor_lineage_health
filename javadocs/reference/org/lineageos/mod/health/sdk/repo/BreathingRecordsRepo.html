Class Overview

Breathing records repository.

This class allows you to retrieve, insert, update and delete breathing records.

Supported records types:

  • InhalerUsageRecord
  • OxygenSaturationRecord
  • PeakExpiratoryFlowRecord
  • RespiratoryRateRecord
  • VitalCapacityRecord

Operations performed in this class should be executed outside of the main / UI thread.
